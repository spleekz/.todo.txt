# .todo.txt

Язык для описания списка дел / идей.

## Специальные символы

`-` — активная задача.

`*` — выполненная задача. ( выделяется табом )

`- * * -` — отделение активных задач от выполненных. ( при желании сворачивается )

После символов задач идёт таб.

## Пример

```
-	Задача 1

-	Задача 2
	многострочная

----- * сделано * -----

	*	Выполненная
		задача 3

```
